# typeScript 中文手册

![logo](/img/hean.png)

创建这个项目的原因是 `typeScript 中文网`已经很久没有更新了，在 Github 搜索 typeScript 相关教程发现大多在线阅读不是没有支持就是已经挂了，所以使用爬虫 + vuePress 技术栈创建了这个仓库

如果喜欢请点击一下 `star` 支持一下
